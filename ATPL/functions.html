There are some rules to declare or call a relaion a function, as the narrator speaks, The rules are as follow:

Rule 1.Existence of the image, if the function is defined as \(f:A -> B\) then \(\forall x \in A\ \exist\ y\ such\ that\ (x,y) \in f\) Means for every element in x there must exist a image or its mapping to the codomain set B

Rule 2. For every input there must be a unique output or an image, this means that no input can have two distinct images! formally speaking if \((x1,y1) \in f\ and\ (x1,y2) \in f\ \implies\ y1=y2\ to\ be\ a\ function\ \)
