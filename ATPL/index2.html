Lets think how this question is made in the first place

Since we know that \(e^x\) is greater than 1 \( \forall x \in (0, \frac{\pi}{2})\) then we can integrate both sides and then we would get something like this. (btw this result could be very easily proven too)

After integrating both sides we would get an inequaltiy we get something interesting as you notice, in the graph that, if we integrate on both the sides from x = 0 to x = X, then we could get something like \(e^x - 1 > x\) and then we can write this as :

\[e^x > 1+x \] \[integrating \ both\ sides\ \] \[e^x > 1+x+\frac{x^2}{2}\]

Since cos is a decreasing function than what we can say

\[cos(e^x) < cos(1+x+\frac{x^2}{2})\]

But what if we could prove Inequalitities thorugh these methods, actually we can so lets look it up But lets build up theory for that

\[to\ prove\ f(x) >= g(x)\] \[ \implies f(x)-g(x)>=0 \] \[let\ h(x) = f(x)-g(x)\]

If we prove that the derivative of h(x) is positve then we can say that h(x) is an increasing function

\[if\ h'(x) >=0 \]

This implies that it is an increasing functions

So then \( \forall x >= a \) we can say that \(h(x)>=h(a)\) See and then this means \(f(x) - g(x) >= f(a)-g(a)\) see, we proved it so easily by just one derivative But what if the differentiation is negative, what that means?

If the differentiation is negative, as you have studied in Application of derivative , if the deriative of the function is negative, which means it is a decreasing function, then from where it is decreasing, lets say from x = a, then \( \forall x >= a: f(x)<=f(a)\)

It is the same as you can say above, so

\[if\ h'(x)<=0 \implies \forall x >= a : h(x)<=h(a)\] \[f(x)-g(x)<=f(a)-g(a)\]

But what about those \(h'(x)\) where we can't tell the differentiation result is positive or negative ,hmm thats simple actually, if we can't see or just tell the difference then we could easily again assume a new function \(j(x) = h'(x)\) and then differentiate the \(j(x)\) function and see if its postive or negative
